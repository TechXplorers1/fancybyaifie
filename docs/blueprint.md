# **App Name**: Esterly Wear Online Store

## Core Features:

- Product Listing and Navigation: Display product listings with categories and filtering based on navigation.
- Search Functionality: Enable users to search for products using keywords with a modal.
- Product Detail Pages: Display detailed information for each product with an option to 'Shop My Outfit'.
- Social Media Integration: Link to Esterly Wear's social media platforms (Instagram, Facebook, Pinterest) in the header and footer.
- Mobile Responsive Design: Ensure the website is fully responsive and adapts to different screen sizes.
- Admin Access: Provide a hidden admin access button in the footer to dispatch a custom admin event, allowing administrators to access the content management system.

## Style Guidelines:

- Primary color: Use a deep, muted blue (#34597A) for a sophisticated and trustworthy feel.
- Background color: Very light desaturated blue (#F0F4F8).
- Accent color: Dusty purple (#8B77A9) for call-to-actions and secondary elements.
- Body font: 'Inter' (sans-serif) for clean and readable text.
- Headline font: 'Belleza' (sans-serif) for fashionable titles and short amounts of text. 'Inter' for body text, if longer text is anticipated.
- Use the icons from 'lucide-react' such as Search, Menu, Instagram, Facebook, and ShoppingBag with consistent styling. Use custom Amazon and Pinterest icons.
- Use a clean and spacious layout with a maximum width of 7xl for main content areas.
